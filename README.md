# 🖼️ Image Resizer

A simple Python tool to resize images. Supports both single image and batch resizing of all images in a folder at once.

## Features
- Resize a single image to any dimension
- Batch resize all images in a folder
- Saves resized images separately (original stays safe)
- Supports JPG, PNG, BMP, WebP formats

## Tech Used
- Python 3.x
- Pillow (PIL)

## Installation
pip install Pillow

## How to Run
python image_resizer.py

## How to Use
1. Run the script
2. Choose option 1 for single image or 2 for batch
3. Enter path and dimensions
4. Done!
